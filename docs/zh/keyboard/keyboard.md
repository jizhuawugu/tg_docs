## 编辑名称（编辑键盘名称）

- 点击"📝 编辑名称" 按钮
- ![img.png](/keyboard/edit_keyboard.png)
- 输入键盘名称（仅支持文本）

## 编辑内容 (编辑点击键盘响应内容)

- 点击"📝 编辑内容" 按钮
- ![img.png](/keyboard/keyboard_response.png)
- 输入键盘响应内容（支持任何格式）

## 按钮管理 (编辑点击键盘响应内容按钮)

- 点击"🔘 按钮管理" 按钮
- <img src="/keyboard/keyboard_button.png" width="250" height="250"/>
- 输入按钮名称和链接  
```md
点击我跳转::https://t.me/xzkb_bot
```
